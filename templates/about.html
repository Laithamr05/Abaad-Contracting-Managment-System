About Abaad Contracting Management System

Overview

The Abaad Contracting Management System (ACMS) is a comprehensive web-based application designed to manage all aspects of a contracting company's operations. The system provides tools for managing branches, employees, departments, projects, clients, suppliers, and materials.

Key Features

  • Branch Management: Manage multiple office locations and branches
  • Employee Management: Track employees, their positions, salaries, and reporting structures
  • Department Management: Organize employees into departments with assigned managers
  • Project Management: Manage projects, their locations, costs, and revenues
  • Client Management: Maintain client information and relationships
  • Material & Supplier Management: Track materials, suppliers, and pricing
  • Work Assignments: Assign employees to projects with roles and hours
  • Advanced Reporting: Generate comprehensive reports and analytics

Technology Stack

  • Backend: Flask (Python web framework)
  • Database: MySQL (InnoDB engine)
  • Frontend: Bootstrap 5, HTML5, CSS3, JavaScript
  • Database Connector: mysql-connector-python

Database Schema

The system uses a relational database design with 10 main entities and proper foreign key relationships. All money fields use DECIMAL(12,2) for precision, and indexes are created on all foreign key columns for optimal performance.

For detailed database documentation, see ERD.md in the docs folder.
